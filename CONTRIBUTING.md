# Contributing to StellarSight

Thank you for your interest in contributing to StellarSight! We welcome contributions from the community.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

## Code of Conduct

Please be respectful and constructive in all interactions.
